Module Description

The HTCondor HDFS daemon is now available as a contrib module and no longer released with the HTCondor binaries. This consists of one daemon, condor_hdfs. It manages the running of the java-based hdfs daemon.

Download

The module is contained in the HTCondor source tree in src/condor_contrib/condor_hdfs as of HTCondor 7.7.5.
