Install a daily build with native packaging

There are several native package repositories containing builds of HTCondor. One is the "daily" repository, which contains a snapshot of the codebase as it stands in the past 24 hours (assuming automated regression testing did not fail). These are not official releases, and are intended just for testing and development.

Install a daily build on Centos 7 

yum -y install https://research.cs.wisc.edu/htcondor/repo/9.1/htcondor-release-current.el7.noarch.rpm
yum -y install condor --enablerepo=htcondor-daily
