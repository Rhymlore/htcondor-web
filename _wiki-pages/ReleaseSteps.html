Milestones in the HTCondor release process:

  1. Development
    • Planning
    • Coding
    • Reviewing
    • Documenting
  2. Code Freeze
  3. Produce release candidate
  4. Update test pools
    • CS pool for stable series
    • NMI + CHTC pools for development series
  5. Testing phase
  6. Igor stress test (in parallel with testing phase)
  7. Documentation complete (in parallel with everything post code freeze)
  8. Release to the web
