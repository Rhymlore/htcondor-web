Module Description

HTCondor Pigeon queues and forwards user log messages via AMQP. It consists of a message broker (a HTCondor daemon) and a message client example implementation (to give you an idea of what's possible).

Download

The module is contained in the HTCondor source tree in src/condor_contrib/condor_pigeon as of HTCondor 7.5.6.

Contact

The contact person for this module is the HTCondor Team (email: htcondor-admin at cs dot wisc dot edu)
